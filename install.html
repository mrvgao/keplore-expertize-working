Installation Steps
1
Download the .vsix file
Click the Download button above for your operating system.
2
Install in VS Code
Open VS Code, press Mac Cmd + Shift + P or Win / Linux Ctrl + Shift + P, then type Install from VSIX and select the downloaded file.
3
Reload VS Code
After installation, reload the window: Cmd/Ctrl + Shift + P → Reload Window.
4
Enter your authorization code
PhysMind will prompt for your authorization code on first launch. Enter the code provided by your administrator (format: pm-cn-...).
5
Start using PhysMind
Click the PhysMind icon in the Activity Bar, create a new task, and start chatting. No additional setup required.
Install via command line
If you have the VS Code CLI available:
code --install-extension physmind-cn-win.vsix
Replace the filename with the version for your platform.
