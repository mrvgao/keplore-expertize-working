Installation Steps
1
Download the .vsix file
Click the Download button above to save physmind-0.2.0.vsix to your computer.
2
Open the Command Palette in VS Code
Mac Press Cmd + Shift + P
Windows / Linux Press Ctrl + Shift + P
3
Run "Extensions: Install from VSIX…"
Type Install from VSIX in the Command Palette, select the command, then choose the downloaded .vsix file. VS Code will install and reload automatically.
4
Set your Anthropic API key
Open VS Code Settings (Cmd+,), search for physmind.apiKey, and paste your key.
Or export it in your shell profile so VS Code picks it up automatically:
export ANTHROPIC_API_KEY="sk-ant-..."
5
Click the PhysMind icon to start
Click the PhysMind icon in the Activity Bar (left sidebar). A chat tab opens in the editor — type your question and press Enter.
Requirements
VS Code
v1.85.0 or later
Visual Studio Code or Cursor
API Key
Anthropic API key
claude.ai/settings → API Keys
CLI (optional)
Claude Code CLI
Required for agentic file editing
Runtime (optional)
Bun
bun.sh — for running CLI source
Install via npm
Install the PhysMind CLI globally via npm. On macOS you may need sudo:
npm install -g @keploreai/physmind
If you get a permissions error on macOS:
sudo npm install -g @keploreai/physmind
Install VS Code extension via CLI
If you have the VS Code CLI available, you can install the extension directly from the terminal:
code --install-extension physmind-0.2.0.vsix
